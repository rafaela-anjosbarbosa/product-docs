# Glossário

- **Tela (Screen)**: visão/feature no front com componentes e comportamento.
- **Componente (UI Component)**: elemento da UI (input, botão, tabela, label, etc.).
- **Requisito (RF)**: comportamento esperado vinculado a uma tela/componente/evento.
- **Regra de Negócio (RN)**: restrição/cálculo/condição que pode valer em UI e/ou API.
- **Fluxo (UC)**: sequência de passos acionada por um evento (ex: clicar salvar).
- **Mensagem (MSG)**: texto padronizado exibido ao usuário.
